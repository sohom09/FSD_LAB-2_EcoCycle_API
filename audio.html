🔍 How the Audio API Works Here

We create an HTMLAudioElement using document.getElementById('ecoAudio'). Key properties used: audio.play(), audio.pause(), audio.currentTime for seeking, audio.duration for total length, audio.volume (0–1), audio.muted, audio.playbackRate for speed, and audio.loop. We listen to timeupdate event to sync the progress bar in real-time.
