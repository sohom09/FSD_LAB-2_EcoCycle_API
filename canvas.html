🔍 How Canvas API Works Here

We call canvas.getContext('2d') to get a drawing context, then use ctx.fillRect() to draw bars, ctx.fillText() for labels, and ctx.createLinearGradient() for the gradient fills. The chart is fully responsive — it resizes with the window.

✏️ Freehand Drawing Pad

Draw anything on the canvas using your mouse or touch screen. Pick a color, adjust the brush size, and express yourself. Works on both desktop and mobile!

🔍 How Freehand Drawing Works

We listen for mousedown, mousemove, and mouseup events (plus touch equivalents). On each move while the button is pressed, we call ctx.lineTo() and ctx.stroke() to draw a continuous path. The eraser mode simply sets ctx.globalCompositeOperation = 'destination-out'.
