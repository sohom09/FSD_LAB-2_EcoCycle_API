# EcoCycle - E-Waste Collection Portal

## Project Overview

EcoCycle is a responsive E-Waste Collection Portal developed using HTML5, Tailwind CSS, JavaScript and HTML5 APIs.

The project promotes sustainable electronic waste management by providing information about recycling, collection services and awareness programs.

## Features

- Responsive Tailwind CSS Design
- Hero Section
- About Section
- Services Section
- Gallery
- Contact Section
- Schedule Pickup Modal

## HTML5 APIs Used

1. Geolocation API
2. Local Storage API
3. File API
4. Canvas API
5. Drag and Drop API
6. Audio API
7. Video API
8. Clipboard API

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- HTML5 APIs
