🔍 How Local Storage API Works Here

The Web Storage API provides localStorage.setItem(key, value) to save data, localStorage.getItem(key) to retrieve it, localStorage.removeItem(key) to delete a key, and localStorage.clear() to wipe everything. Data is stored as strings — objects must be serialized with JSON.stringify() and parsed with JSON.parse(). Storage limit is typically 5–10 MB per origin.
