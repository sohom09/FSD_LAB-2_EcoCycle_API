🔍 How the WebSocket API Works Here

We create a connection with const ws = new WebSocket('wss://echo.websocket.events'). The four key event handlers are: ws.onopen (connection established), ws.onmessage (data received), ws.onerror (error occurred), and ws.onclose (connection closed). We send data with ws.send(JSON.stringify(data)) and close gracefully with ws.close(). WebSocket enables true full-duplex communication — unlike HTTP, the server can push data to the client at any time without a new request.
