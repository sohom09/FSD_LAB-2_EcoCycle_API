📄 Read Text File

Upload a plain text file (like a pickup manifest or recycling record) and read its contents using FileReader.readAsText().

🔍 How File API Works Here

The File API gives us the FileList from <input type="file"> or drag events. For each File object we create a new FileReader(). reader.readAsDataURL(file) gives a base64 string for image preview. reader.readAsText(file) reads text content. The reader.onload callback fires when reading is complete — all processing happens locally in the browser, with zero data sent to any server.
