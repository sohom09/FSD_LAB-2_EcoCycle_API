🔍 How Geolocation API Works Here

We use navigator.geolocation.getCurrentPosition(success, error) to get a one-time fix, and navigator.geolocation.watchPosition() for live tracking. The browser prompts the user for permission — no location data is sent to any server. The returned GeolocationPosition object contains coords.latitude, coords.longitude, and coords.accuracy in metres.
