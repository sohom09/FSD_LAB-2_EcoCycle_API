🔍 How the Clipboard API Works Here

The Async Clipboard API uses navigator.clipboard.writeText(text) to write to the clipboard and navigator.clipboard.readText() to read from it. Both return Promises. Writing typically works without permission, but reading requires the user to grant clipboard-read permission. We use .then() and .catch() to handle success and failure gracefully. This is far more reliable than the older document.execCommand('copy') approach.
