🔍 How Drag & Drop API Works Here

Each item has draggable="true" and an ondragstart handler that calls event.dataTransfer.setData('text', itemId) to store the item's ID. The bins have ondragover (calls event.preventDefault() to allow drops) and ondrop handlers that read the item ID with event.dataTransfer.getData('text'), then check if it's correct e-waste. We also handle ondragleave to remove the visual highlight when you move away.
